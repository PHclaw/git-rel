# git-rel

> Generate changelogs from GitHub releases and merged PRs.

```bash
git-rel open-webui/open-webui
git-rel mem0ai/mem0 --format json
git-rel PHclaw/git-rel --since v0.1.0
```

## Features

- Fetches latest release info from any public GitHub repo
- Aggregates merged PRs between releases
- Categorizes changes by type: breaking, feature, bugfix, docs, refactor, i18n, etc.
- Output formats: `markdown` (default), `json`, `text`

## Installation

```bash
pip install git-rel
```

Or from source:

```bash
pip install -e .
```

## Usage

```bash
git-rel owner/repo                    # Markdown changelog
git-rel owner/repo --format json      # JSON output
git-rel owner/repo --format text      # Plain text
git-rel owner/repo --since v1.2.0     # From specific tag
git-rel owner/repo --verbose          # Debug info
```
